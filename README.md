# AI vs Human Text Analysis Dashboard

This project analyzes differences between AI-generated and human-written text using Power BI.

## Dataset
- Text length (characters & words)
- Quality score
- Plagiarism score
- Sentiment
- Topic distribution
- Time-based growth

## Dashboard Highlights
- AI vs Human content distribution
- Quality and plagiarism comparison
- Sentiment analysis
- Topic-wise comparison
- Growth trends over time

## How to View
- Download the `.pbix` file and open it using **Power BI Desktop**
- Screenshots are provided for quick preview
